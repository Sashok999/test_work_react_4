# Приложение "Мои фильмы"

React-приложение для управления списком любимых фильмов.

## Функциональность

- Отображение списка фильмов с названием, годом выпуска и жанром
- Добавление/удаление фильмов в избранное
- Счетчик избранных фильмов
- Отдельный список избранных фильмов

## Технологии

- React (функциональные компоненты, хуки)
- Tailwind CSS для стилизации

## Особенности
- Адаптивный дизайн
- Интерактивные кнопки добавления в избранное
- Динамическое обновление списка избранных

Я использовал React с хуками, чтобы сделать интерактивное приложение для добавления фильмов в избранное.
Для красивого внешнего вида применил Tailwind CSS, который легко настраивается через классы.
